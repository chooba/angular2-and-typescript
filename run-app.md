# Запуск программы #

## Перейти в основную папку ##

В терминале набрать:

    cd ~/dev/chooba/angular2-and-typescript

## Перейти в папку программы ##

Затем выполнить:

    cd chooba-app

## Запуск программы ##

Запустить:

    ng serve

## Открыть в браузере ##

[http://localhost:4200](http://localhost:4200)
